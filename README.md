# Interactive JSON Editor

An interactive React component for editing JSON Components.

## Installation

```bash
npm install interactive-json-editor
```

## Usage

```jsx
import React from 'react';
import JsonEditor from 'interactive-json-editor';

const App = () => {
  return (
    <div>
      <h1>My App</h1>
      <JsonEditor />
    </div>
  );
};

export default App;
```
